B. Questions

Q1. What happens if I remove glEnable(GL_DEPTH_TEST)?

Without the depth test, OpenGL draws faces only in the order they are rendered. Back faces may appear in front, and the cube looks incorrect because the depth relationships are not handled.

Q2. Exit when the Q key is pressed. What did I change?

I added this code to the event loop:

if (windowEvent.type == SDL_EVENT_KEY_UP &&
    windowEvent.key.key == SDLK_q) {
    quit = true;
}

Q3. Make the light come from below in CubeLit.

I changed the light direction in the fragment shader: 

const vec3 lightDir = vec3(0, -1, 1);

Q4. Custom triangle model

I created a file named triangle.txt with one large triangle.

std::string file_name = "models/triangle.txt";

B. Questions

Q1. What happens if I remove glEnable(GL_DEPTH_TEST) in Cube3D?

When the depth test is removed, OpenGL no longer checks which fragment is closer to the camera. Faces are drawn only in the order they are rendered. As a result, some back faces may appear in front of nearer faces, and the cube looks wrong and flat because the occlusion is incorrect.

Q2. Exit when the “Q” key is pressed. What did I change?

In the SDL event loop I added an extra key check for the Q key: 

if (windowEvent.type == SDL_EVENT_KEY_UP &&
    windowEvent.key.key == SDLK_q) {
    quit = true;
}

Now the program exits when I press either Esc or Q.
